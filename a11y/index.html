aXe

  • Most conservative
  • Cites WCAG and explains violations
  • Chrome or Firefox extension

Lighthouse

  • Also tests performance and best practices
  • Only available in Chrome DevTools

Wave

  • Visually marks violations
  • Best outline
  • Chrome or Firefox extension

Koa11y

  • Most far-reaching
  • Cites WCAG
  • Potential for false positives/judgement calls
  • Standalone application

tota11y

  • Visually marks violations
  • Explains violations
  • Bookmarklet
