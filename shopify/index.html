Other Integrations

The Pass-Through

  • If user is logged into American Ancestors
    • User is logged into Shopify
    • Redirected to product page
  • If user is not logged into American Ancestors
    • User is redirected to product page with query string ?pass-through=true appended
    • User will be directed to log in or create an account if they attempt to checkout
